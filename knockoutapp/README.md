
# Using generator-ko-ts2

##### Includes

* Knockout MVVM

* Typescript

* Bootstrap 4 alpha

* Sass - for css

* Browsersync - syncing changes in browser


#### Generate project

        yo ko-ts2

#### Build typescript

        tsc

#### Serve commands
    
    gulp dist
    gulp src

#### Generate components
        

        yo ko-ts2:component componentName
